#### comment-info：为代码添加头部注释信息

#### 使用
##### 安装

1. `git clone https://github.com/AHAOAHA/comment-info.git`
2. `cd comment-info`
3. `./install.sh`

##### 卸载
1. `cd [your clone path]/comment-info`
2. `./uninstall.sh`

#### 当前支持

- C
- C++
- bash
- Go
- Vimscript
